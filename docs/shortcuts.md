# Keyboard shortcuts

| Action | Shortcut |
| --- | --- |
| Create new project | `Ctrl+N` |
| Save project | `Ctrl+S` |
| Save project as | `Ctrl+Shift+S` |
| Open project | `Ctrl+O` |
| Close project | `Ctrl+W` |
| Run analysis | `Ctrl+R` |
| Exit application | `Ctrl+Q` |

Shortcuts may behave differently while a text field or operating-system dialog has focus.
