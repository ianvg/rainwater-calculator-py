# Bootstrap Icons attribution

The system builder renders the original **Cloud Rain** SVG for its rainwater input.
It is stored as `cloud-rain.svg`.

The builder also uses these unmodified Bootstrap Icons SVGs:

- **Filter Square** for filtration systems
- **Buildings Fill** for municipal water backup
- **House Door Fill** for end uses
- **Funnel Fill** for first-flush diversion
- **Box Arrow Down Right** for overflow discharge

**Arrow Right Circle Fill** and **Speedometer2** are retained as unused legacy
candidates. Transfer and booster pumps now use the selected Material Design
Icons **pump** artwork.

**Cloud Rain Heavy** is retained as `cloud-rain-heavy.svg` for possible use in a future
weather, rainfall-intensity, or other visual feature. It is not currently displayed by the app.

- Cloud Rain Heavy: https://icons.getbootstrap.com/icons/cloud-rain-heavy/
